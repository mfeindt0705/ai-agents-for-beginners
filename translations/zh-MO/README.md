## 🌱 開始

本課程包含介紹建立AI代理基本原理的課程。每節課涵蓋其主題，您可從任意一課開始學習！

本課程提供多語言支援。請前往我們的[語言選擇頁面](#-multi-language-support)。

若您是第一次使用生成式AI模型，請參考我們的[生成式AI初學者課程](https://aka.ms/genai-beginners)，其中包含21課如何使用生成式AI建構應用。

別忘了[為本倉庫點星 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 以及 [分叉本倉庫](https://github.com/microsoft/ai-agents-for-beginners/fork) 來執行程式碼。

### 交流學習者、獲得您的問題解答

如果您卡住，或有任何關於建立AI代理的問題，歡迎加入我們專屬的 Discord 頻道：[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)。

### 您需要準備的東西

課程中的每課都包含範例程式碼，這些程式碼位於 code_samples 資料夾中。您可以[分叉本倉庫](https://github.com/microsoft/ai-agents-for-beginners/fork)以建立自己的副本。

這些程式碼範例使用 Microsoft Agent Framework 以及 Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) — 需要 Azure 帳戶

本課程使用以下微軟AI代理框架與服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

欲了解本課程程式碼的執行方式，請見 [課程設定說明](./00-course-setup/README.md)。
