## 🌱 入門指南

本課程包含關於建立 AI 代理的基礎課程。每堂課均針對不同主題，您可以從任何一堂課開始學習！

課程支援多種語言。請參閱我們的[可用語言列表](#-multi-language-support)。

若您是第一次使用生成式 AI 模型，請參考我們的[生成式 AI 初學者課程](https://aka.ms/genai-beginners)，內含 21 堂關於生成式 AI 建置的課程。

別忘了給本儲存庫[星標 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)並且[分叉此儲存庫](https://github.com/microsoft/ai-agents-for-beginners/fork)來執行程式碼。

### 認識其他學習者，獲得問題解答

如果您遇到困難或對於建立 AI 代理有任何疑問，請加入我們在 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) 的專屬 Discord 頻道。

### 您所需的條件

本課程每堂課均有程式碼範例，可於 code_samples 資料夾內找到。您可以[分叉此儲存庫](https://github.com/microsoft/ai-agents-for-beginners/fork)來建立您自己的副本。

本課程中的程式碼範例使用 Microsoft 代理框架搭配 Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 帳戶

本課程使用微軟提供的以下 AI 代理框架與服務：

- [Microsoft 代理框架 (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


欲瞭解教學程式碼的執行詳情，請參閱 [課程設定](./00-course-setup/README.md)。
