## 🌱 البدء

تحتوي هذه الدورة على دروس تغطي أساسيات بناء وكلاء الذكاء الاصطناعي. كل درس يغطي موضوعه الخاص لذا ابدأ من أي مكان تريده!

هناك دعم متعدد اللغات لهذه الدورة. اذهب إلى [اللغات المتوفرة هنا](#-multi-language-support).

إذا كانت هذه هي المرة الأولى التي تبني فيها باستخدام نماذج الذكاء الاصطناعي التوليدية، تحقق من دورة [الذكاء الاصطناعي التوليدي للمبتدئين](https://aka.ms/genai-beginners)، والتي تتضمن 21 درسًا حول البناء باستخدام GenAI.

لا تنسَ [وضع نجمة (🌟) على هذا الريبو](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) و [انشاء فرع (fork) لهذا الريبو](https://github.com/microsoft/ai-agents-for-beginners/fork) لتشغيل الكود.

### تعرف على المتعلمين الآخرين، واحصل على إجابات لأسئلتك

إذا واجهت صعوبة أو كانت لديك أي أسئلة حول بناء وكلاء الذكاء الاصطناعي، انضم إلى قناتنا المخصصة في ديسكورد داخل [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### ما تحتاجه

يتضمن كل درس في هذه الدورة أمثلة تعليمية يمكنك العثور عليها في مجلد code_samples. يمكنك [انشاء فرع (fork) لهذا الريبو](https://github.com/microsoft/ai-agents-for-beginners/fork) لإنشاء نسختك الخاصة.

تستخدم أمثلة الكود في هذه التدريبات إطار عمل Microsoft Agent مع خدمة Azure AI Foundry Agent V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - يتطلب حساب Azure

تستخدم هذه الدورة الأطر والخدمات التالية لوكلاء الذكاء الاصطناعي من مايكروسوفت:

- [إطار عمل Microsoft Agent (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [خدمة Azure AI Foundry Agent V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

لمزيد من المعلومات حول تشغيل الكود لهذه الدورة، اذهب إلى [إعداد الدورة](./00-course-setup/README.md).
